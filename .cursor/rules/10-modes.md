# Modes
- NORMAL MODE (default):
  - Minimal diffs
  - Up to 3 files
  - Plan → diff → stop

- BULK MODE (explicitly requested):
  - Up to 25 files
  - Allowed only on staging branch
  - Must include a rollback plan
  - Show per-file diffs, then stop

- Always provide a short manual verification checklist I can do in Shopify Preview.
